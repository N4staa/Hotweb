Web Developer

Qualifications

LEVEL 2 IT USER SKILLS

LEVEL 3 IN WEB DEVELOPMENT

LEVEL 4 MATHEMATICS

WEB DEVELOPMENT DEGREE IS ESSENTIAL.

You will need :

  1. knowledge of computer operating systems, hardware and software.
  2. to be thorough and pay attention to detail.
  3. the ability to write computer programs.
  4. to be flexible and open to change.

Responsibilities

  1. Converting design mockups and wireframes into functional web pages using HTML, CSS, and JavaScript.
  2. Ensuring websites work well on various devices and screen sizes.
  3. Creating and enhancing interactive elements to improve user experience.
  4. Ensuring websites work consistently across different web browsers.

IT Technician

Qualifications

BACHELOR DEGREE

A LEVEL COMPUTING & IT

LEVEL 3 COMPUTER SCIENCE

LEVEL 3 SYSTEM ANALYSIS

You will need :

  1. To be able to diagnose and solve computer problems, technicians often gather data by various means.
  2. Once they have gathered the data, they use this to discover where problems originate and devise solutions.
  3. they can diagnose faults and implement fixes.
  4. Strong problem-solving abilities are required .

Responsibilities

  1. Providing technical assistance and support for incoming queries and issues related to computer systems, software, and hardware.
  2. Diagnosing and resolving technical hardware and software issues.
  3. Assisting users with the setup, maintenance, and troubleshooting of IT equipment, including computers, printers, and mobile devices.
  4. Installing and configuring new hardware, including computers, servers, routers, and other IT equipment.
